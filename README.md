# Workout Tracker — Telegram Mini App

Личный трекер тренировок. 8-недельная программа, 3 дня в неделю. Без бэкенда, без аккаунтов.

---

## Стек и хостинг

- Чистый HTML/CSS/JS — никакого build-шага
- Хостинг: **GitHub Pages** → бесплатный URL вида `https://имя.github.io/репо`
- Прогресс: **localStorage** (сохраняется между сессиями)
- Открытие: **Telegram Mini App** через бота

---

## Что умеет приложение

- Автоматически открывает следующую незавершённую тренировку
- Для каждого упражнения — ряд кружков (один на подход). Тап = выполнено, ещё тап = снять отметку
- Дропсеты: 3 кружка + текст `12-6-6` как подсказка
- Таймер отдыха 2 мин — запускается вручную, можно прервать досрочно. SVG-кольцо убывает, краснеет при < 30 сек. Вибрация по окончании (Telegram HapticFeedback)
- Когда все кружки заполнены → появляется "Завершить тренировку"
- После завершения: вибрация + "Готово!" на 2 сек → следующая тренировка
- После 3 дней недели → автопереход на следующую неделю
- После недели 8 → "Программа завершена" + кнопка "Начать заново"
- Состояние тренировки сохраняется: закрыл приложение посередине → продолжаешь с того места

## Дизайн — "Iron Log"

### Палитра
| Имя | Hex | Применение |
|-----|-----|------------|
| near-black | `#0B0B10` | фон |
| surface | `#13131C` | карточки упражнений |
| iron-red | `#C0391E` | тяжёлая интенсивность |
| steel-blue | `#2870B8` | средняя интенсивность |
| brass | `#B8903A` | выполненные подходы, кольцо таймера |
| green | `#39875A` | кнопка "Завершить тренировку" |

### Ключевые решения

**Левая граница карточки** — 4px полоска передаёт интенсивность без текстового бейджа. Красная = тяжёлая, синяя = средняя, латунная = всё выполнено (карточка тускнеет).

**SVG-кольцо таймера** — убывающее кольцо вокруг цифры вместо прогресс-полоски. При ≤ 30 сек кольцо краснеет.

**Eyebrow-порядок в карточке** — `3×8–12` стоит над названием упражнения: объём виден раньше, чем читаешь название.

**Шрифт Oswald 700** — для названий дней. Конденсированный, промышленный. Всё остальное — `system-ui`.

**Прогресс дней** — три горизонтальные полоски под названием дня: серая (впереди), белая (сейчас), латунная (сделано).

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub
2. Закинь файлы: `index.html`, `data.js`, `app.js`, `style.css`, `.nojekyll`
3. `Settings → Pages → Deploy from branch: main, / (root)`
4. Получишь URL — подожди 1–2 минуты пока GitHub соберёт

---

## Настройка Telegram Mini App

1. Напиши [@BotFather](https://t.me/BotFather) в Telegram
2. `/newbot` → придумай имя и username боту
3. `/setmenubutton` → выбери бота → вставь URL из GitHub Pages
4. Открывай через кнопку меню в боте

## localStorage — структура данных

```js
{
  week: 0,       // 0–7, текущая неделя
  day: 0,        // 0–2, текущий день (пн/ср/пт)
  workout: {     // null если день не начат
    week: 0,
    day: 0,
    sets: [      // per exercise → per set → boolean
      [false, false, false, false, false, false],
      ...
    ]
  }
}
```

Ключ в localStorage: `wt_v1`

## Программа (из xlsx)

8 недель, 3 дня в неделю (Пн / Ср / Пт):

- **Понедельник** — бицепс, трицепс, плечи (6 упражнений)
- **Среда** — базовые: жим, тяга блока, присед, румынская тяга (4 упражнения)
- **Пятница** — бицепс, трицепс, плечи, другая вариация (6 упражнений)

Интенсивность чередуется по неделям (средняя → тяжёлая → средняя...), объём растёт от 3 до 7 подходов к концу программы.
